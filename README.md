This project is a Next.js (App Router) dashboard application that includes:

- Authentication using NextAuth
- State management using Zustand
- Client-side caching for Users and Products
- MUI-based UI with responsive tables, cards, and tabs
- Protected routes using Next.js Middleware

## Features
- Login with DummyJSON API  
- Protected dashboard (`/dashboard/*`)  
- Users listing with pagination, search, and caching  
- Products listing with pagination, search, category filter, and caching  
- Product detail page  
- Persistent dashboard tabs   
- Responsive UI using MUI

## Setup Instructions

### Clone the repository
git clone https://github.com/shashi9027/green-assign.git
cd green-assign

### Install dependencies
npm install

### Run the project 
npm run dev
